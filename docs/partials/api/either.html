either

Description

Возвращает второй аргумент, если он != null. Иначе возвращает первый аргумент. Используется для программирования в функциональном стиле.

Функция обёрнута в curry

Usage

either(def, val);

Parameters

ParamTypeDetails
defany

значение по-умолчанию

valany

значение для сравнения с null

Returns

any

возвращает val, если val != null. Иначе возвращает def
